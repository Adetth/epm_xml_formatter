v1.0.0 - "Major UI Update"
- UI REWRITE: Transformed the tool into a desktop-class Web-App featuring sidebar navigation, glassmorphism frosted panels, Fira Code/Outfit typography, and an animated ambient background.
- WYSIWYG GRID: The Grid Visualizer is now fully interactive. Click any cell to open a custom, draggable in-browser Hex Editor with live preview and quick-select preset swatches.
- STATE MANAGEMENT: Added an Undo/Redo memory stack to instantly revert accidental formatting modifications.
- BROWSER PERSISTENCE: Workspaces now auto-save to LocalStorage. Accidentally refreshing or closing the tab will no longer wipe loaded RAM files.
- SYNTAX HIGHLIGHTING: Replaced standard textareas with CodeMirror to provide VS-Code style auto-coloring and line numbers in the Raw Text tab.
- THEME PORTABILITY: Added JSON Import/Export capabilities to save, share, and re-apply custom EPM color dictionary palettes.
- RENDERER UPGRADE: The Python engine now extracts 'txtColor' and automatically calculates high-contrast text fallbacks for better visual accessibility.
- BUG FIXES: Aggressively scoped Hex input CSS to prevent overriding standard text fields, and implemented WebKit hacks to hide unsightly scrollbars while retaining scroll functionality.
